Website Design & Project Managment | 2019

LIVE WEBSITE

PROJECT OVERVIEW

What

Winkedin is a dating app that allows college students to connect with each other. Winkedin does so by letting users pick and choose potential candidates by fulfilling their specific criteria.

Why

To complete a projects use waterfall methodology

Who

My role in this project was project manager and web developer

Tool

HTML, CSS, Javascript, Bootstrap, Adobe Xd, Gantt Chart

When

March 2020

PROCESS

PHASE ONE

Planning

When thinking about what features to add to the website, I sat down and did some research on the current market. After analyzing the good and the bad, I decided to sketch out how we wanted the website to look like. After feedback and many iterations, this is the final low fidelity mockup I came up with.

PHASE TWO

Research

I decided to use bootstrap for this project as it allowed me to quickly render the look and feel of the website. Time is an extremely important aspect of the duties of a web developer and I felt that by using bootstrap , I can have a working model with little to no time.

PHASE THREE

Testing

After completing the website, my next task was to make sure the website was responsive on all viewports. Luckily , bootstrap has a built in responsive framework which saves users a great amount of time when it comes to making a responsible website.

PHASE FOUR

REFLECTION

Winkedin was a great project which allowed me to play with the CSS framework bootstrap. As a member in a big team, we often went back and forth on the design and layout of a website and having bootstrap made making changes faster and easier.
