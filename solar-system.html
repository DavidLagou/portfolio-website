SOLAR SYSTEM

Website Design | 2019

LIVE WEBSITE

PROJECT OVERVIEW

What

This website is all about the solar system. I wanted to share my deep interest with astrology. The website focuses on comparing and contrasting the different planets , moons and dawf planets that are found within the milky way galaxy

Why

To understand the inner workings of wordpress.

Who

The project was designed and coded by myself.

Tool

WordPress , Adobe Photoshop

When

March 2020

PROCESS

PHASE ONE

Planning

I spent some time figuring out what I wanted to design and code. This was a hard one because this was going to be my first ever website that I create in the program so I wanted it to be a good one. After finding my topic, I then searched through Nasa’s website and gathered all of the information that could be used in order to create the website.

PHASE TWO

Mockups

I then spent quite a bit of time coming up with low fidelity wireframes for how I wanted to position the text/ images. This took the majority of the time because I needed to pick a wordpress template that allowed for quite a bit of flexibility. My main goal was to make the website functional across all devices, whether it be an iPhone or an iPad, the website needed to be 100% responsive. I found myself getting lost in exploring all of the different options.

PHASE THREE

Application

With all of the assets ready, I went ahead and started adding all of the information into the wordpress website. After playing around with the page builder I felt confident to finalize the design.

PHASE FOUR

Testing

Making sure my website reacts the same on multiple browsers is a vital aspect to every website I create. I did testing on browsers like Firefox, safari and chrome and made sure there weren’t any discrepancies in the sites functionality or design.

PHASE FIVE

REFLECTION

Looking back, I found this project to be beneficial in getting me started with wordpress. Its GUI makes adding content extremely easy and intuitive. Although hand coding websites give me a lot more flexibility , I can appreciate the speed in which I could generate a mockup into a fully functioning website. With wordpress powering about a third of the websites on the web, being able to understand how it works is a key asset to have when building for the web.
